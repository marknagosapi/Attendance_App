
How To Run:
    - Install Node, follow this tutorial: https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
    - Install yarn with node: npm install --global yarn
    - Copy The Repository onto your computer
    - When In /Client Folder enter in terminal: npm install 
    - After that start the app with typing in terminal: yarn start 
--------------------------------------------------------------------------------------------------------------------------------
